# 그리디 알고리즘

기준에 제일 잘 만족하는 거부터 사용하는 유형</br>
문제에서 '가장 큰 순서대로'같이 기준을 제시해줌</br>
정렬 알고리즘과 짝을 이루어 출제됨</br>
대부분 문제에선 최적의 해를 찾아줄 수 없음(정당한지 한번쯤 생각해보자)</br>
